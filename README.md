# Bem vindo ao meu projeto
## Descrição
Este projeto é uma simples implementação de um backend que controla um mensageiro de textos básico. Veja os requistos funcionais ao fim do documento.

## Instalação

```bash
$ npm install
```

## Rodando a aplicação na sua máquina

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
